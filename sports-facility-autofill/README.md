# Sports Facility Auto-fill

スポーツ施設予約フォームを自動入力するChrome拡張機能です。

## 概要

この拡張機能は、長野技術科学大学のスポーツ施設予約システム（https://nut-sports.revn.jp/）の予約フォームを自動的に入力します。
毎回同じ情報を手動で入力する手間を省き、予約プロセスを効率化します。

## 機能

- **基本情報の自動入力**
  - 団体名
  - 氏名（姓・名）
  - 学籍番号
  - 電話番号
  - メールアドレス

- **予約詳細の自動入力**
  - 所属・学年
  - 予約時間
  - 活動名称・目的
  - 使用設備
  - 参加人数
  - 学外者情報
  - 備考
  - 利用規約への同意

- **設定の保存**
  - Chrome Sync Storageを使用して設定を保存
  - 一度設定すれば、他のデバイスでも同期可能

## インストール方法

1. このリポジトリをクローンまたはダウンロードします
   ```bash
   git clone <repository-url>
   cd sports-facility-autofill
   ```

2. Chromeで拡張機能管理ページを開きます
   - `chrome://extensions/` にアクセス

3. 右上の「デベロッパーモード」を有効にします

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. このプロジェクトのディレクトリを選択します

## 使い方

### 1. 初期設定

1. Chrome拡張機能アイコンをクリックして、拡張機能のポップアップを開きます
2. 自動入力したい情報を各フィールドに入力します
3. 「Save Settings」ボタンをクリックして設定を保存します

### 2. 自動入力の実行

1. スポーツ施設予約ページ（https://nut-sports.revn.jp/reservations/add*）にアクセスします
2. ページが読み込まれると、保存された情報が自動的にフォームに入力されます
3. 必要に応じて情報を確認・修正し、予約を完了します

## プロジェクト構成

```
sports-facility-autofill/
├── manifest.json       # 拡張機能の設定ファイル
├── popup.html         # 設定画面のUI
├── popup.js           # 設定画面のロジック
├── content.js         # フォーム自動入力のロジック
├── images/            # 拡張機能のアイコン
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md          # このファイル
```

### ファイル説明

- **manifest.json**: Chrome拡張機能のメタデータと権限を定義
- **popup.html**: ユーザーが設定を入力するためのポップアップUI
- **popup.js**: 設定の保存と読み込みを処理
- **content.js**: 予約フォームページに注入され、自動入力を実行

## 技術仕様

- **Manifest Version**: 3
- **権限**:
  - `storage`: 設定の保存
  - `activeTab`: アクティブタブへのアクセス
  - `scripting`: コンテンツスクリプトの実行
- **対象URL**: `https://nut-sports.revn.jp/*`

## 開発

### 前提条件

- Google Chrome ブラウザ
- 基本的なHTML/CSS/JavaScriptの知識

### デバッグ

1. `chrome://extensions/` で拡張機能の「詳細」をクリック
2. 「バックグラウンドページを検証」または「ビューを検証」でDevToolsを開く
3. コンソールログで動作を確認

### カスタマイズ

フォームフィールドを追加・変更する場合:

1. `content.js`の`FIELD_IDS`オブジェクトに新しいフィールドIDを追加
2. `popup.html`に対応する入力フィールドを追加
3. `popup.js`で設定の保存・読み込みロジックを更新

## 注意事項

- この拡張機能は長野技術科学大学のスポーツ施設予約システム専用です
- フォームの構造が変更された場合、拡張機能の更新が必要になる可能性があります
- 自動入力された情報は必ず確認してから送信してください
