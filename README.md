# Google Antigravity

Google Antigravityは、新しい開発手法を構築するために設計されたAI搭載のコーディングアシスタントです。強力なエージェントAIをVS Codeベースのエディタに統合し、コードの記述、複雑なプロジェクトの理解、タスクの自動化を支援します。

## 主な機能

### 1. コアエージェント
Antigravityの中核となるのは、最先端のモデル（Gemini 3 Pro、Claude Sonnet 4.5、GPT-OSSなど）を搭載した多段階推論エージェントです。以下のことが可能です：
- **推論と計画:** 複雑な問題を管理可能なステップに分解します。
- **ツールの使用:** ターミナルコマンドの実行、ファイルの編集、コードベースの検索を行います。
- **コミュニケーション:** タスク、計画、成果物（アーティファクト）を通じて対話します。

### 2. 統合エディタ
堅牢なVS Codeコードベース上に構築されたエディタは、AIによって強化された使い慣れた環境を提供します：
- **AI提案:** 「Tab」キーによるインテリジェントなコード補完と生成。
- **エージェント統合:** クイックタスク用の「Command」インターフェースや、より大きな目的のための「Agent Side Panel」を通じて、エージェントとシームレスに連携します。
- **拡張機能:** Open VSXマーケットプレイスをサポートしています。

### 3. ブラウザサブエージェント
Webをナビゲートし、対話できる専門のエージェントです：
- **Webインタラクション:** Webサイトのクリック、スクロール、入力、データ抽出を行います。
- **視覚的理解:** スクリーンショットやDOM要素をキャプチャしてWebページを理解します。
- **記録:** 検証やレビューのためにブラウザセッションを記録します。

### 4. Model Context Protocol (MCP)
エージェントをより広範なエコシステムに安全に接続します：
- **コンテキストリソース:** ローカルツール、データベース（Neon、Supabaseなど）、サービス（Linear、Notion、GitHubなど）からデータを読み取ります。
- **カスタムツール:** エディタから直接外部システムのアクションを実行します。

### 5. アーティファクト
エージェントはコラボレーションを促進するために構造化された出力を生成します：
- **ドキュメント:** 計画、タスク、ドキュメント用のMarkdownファイル。
- **コード:** 差分ブロックとファイル編集。
- **メディア:** 画像、図、ブラウザの記録。

### 6. ナレッジシステム
作業から学習する永続的なメモリ層です：
- **自動キャプチャ:** 洞察、パターン、解決策を「ナレッジアイテム」として自動的に保存します。
- **再利用:** 学習した知識を将来のタスクや会話に適用します。
